This repo contains example of AWS lambda use.
You will need to have localstack installed on your computer to test if it works. - can be installed on linux via "sudo pip install localstack"

To start:
- Copy all files into one folder on your computer
- start localstack server
- run lambda function

Expected result:
Creation of s3 bucket "bucketerie"
Creation of json file "colors_united.json"
json file visible inside s3 bucket
